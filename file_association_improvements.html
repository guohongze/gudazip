🔧 具体改进内容

  • 1. 智能文件关联逻辑

    安装时行为：如果用户选择关联文件，默认关联所有可处理的文件类型

    手动选择：用户在设置中勾选特定文件类型后，系统只关联选定的文件

    ❌ 改进前

    • 用户需要逐个选择文件类型
    • 容易遗漏常用格式
    • 设置过程繁琐

    ✅ 改进后

    • 安装时自动关联所有格式
    • 用户可精确控制关联
    • 设置过程简化

  • 2. 移除冗余的"设置为默认压缩软件"选项

    该选项功能与文件关联重复，通过全选/反选文件类型即可实现相同效果

    # 移除的代码片段 self.set_as_default_cb = QCheckBox("将GudaZip设置为默认压缩软件") # 该功能现在通过文件关联的全选/反选来实现
    已移除

  • 3. 启动时默认压缩软件检测

    程序启动时自动检测是否为默认压缩软件，如果不是则提供友好的设置对话框

    ❌ 改进前

    • 用户不知道当前关联状态
    • 需要手动进入设置查看
    • 设置过程不够直观

    ✅ 改进后

    • 启动时自动检测关联状态
    • 提供一键设置选项
    • 支持"不再提示"选项
    新增功能

  • 4. 优化状态显示样式

    文件关联和右键菜单状态提示改为正常字体和大小，提升可读性

    ❌ 改进前

    self.association_status_label.setStyleSheet( "color: #666; font-size: 9pt; font-style: italic;" )

    ✅ 改进后

    self.association_status_label.setStyleSheet( "color: #666; padding: 8px; " "background-color: #f8f9fa; border-radius: 4px;" )
    样式优化

  • 5. 增强的文件关联管理

    新增 get_associated_extensions() 方法，实时获取当前已关联的文件类型

    def get_associated_extensions(self) -> List[str]: """获取当前已关联的扩展名列表""" associated = [] if not self.registry.is_available(): return associated for ext in self.supported_extensions: try: if self.registry.check_file_association(ext, self.prog_id): associated.append(ext) except Exception: continue return associated
    新增方法
