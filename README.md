# GudaZip - Python桌面压缩管理器

## 项目简介

GudaZip是一个基于Python和PySide6开发的桌面压缩文件管理器，提供类似资源管理器的界面，支持多种压缩格式的浏览、解压、压缩等操作。

## 功能特性

### 当前版本 (v0.2.1)
- ✅ 资源管理器风格的用户界面
- ✅ 🆕 美化的快捷操作面板（方块按钮 + 图标）
- ✅ 🆕 现代化工具栏设计
- ✅ 左侧文件系统导航树
- ✅ 右侧多标签文件查看器
- ✅ ZIP文件的完整支持（创建、解压、查看）
- ✅ 压缩包创建功能（支持多文件、文件夹、密码保护）
- ✅ 压缩包解压功能（支持完整解压和选择性解压）
- ✅ RAR文件的只读支持
- ✅ 完整的菜单、工具栏和快捷键

### 支持的压缩格式
- **ZIP** - 完整支持（创建、解压、查看、密码保护）
- **RAR** - 解压支持（需要rarfile库）
- **7Z** - 解压支持（需要py7zr库）
- **TAR/GZ** - 解压支持

## 开发者注意事项

### RAR支持说明
RAR格式支持需要：
1. 安装`rarfile`库: `pip install rarfile`
2. 安装WinRAR或unrar命令行工具
3. 确保unrar.exe在系统PATH中或在标准安装位置

### 代码规范
- 使用UTF-8编码
- 遵循PEP8代码规范
- 类和函数需要中文注释
- 采用Model-View架构

## 快速开始

### 1. 安装运行
```bash
# 克隆项目
git clone https://github.com/yourusername/gudazip.git
cd gudazip

# 安装依赖
pip install -r requirements.txt

# 运行程序
python main.py
```

### 2. 创建压缩包
- 按 `Ctrl+N` 或点击"新建压缩包"
- 选择要压缩的文件和文件夹
- 设置压缩选项和保存路径
- 点击"创建压缩包"

### 3. 解压压缩包
- 在文件浏览器中选择压缩包文件
- 按 `Ctrl+E` 或点击"解压到..."
- 在解压对话框中设置解压选项
- 选择解压路径和文件
- 点击"开始解压"
